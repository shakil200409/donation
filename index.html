Important FAQ

What is the Document Object Model (DOM)?

All HTML elements as Objects, we can find element(s) in JavaScript by DOM. It is the layout structure of elements in a file.

How do you select an element from the DOM?

We can find the elements in 4 ways

  • getElementById (Mostly Used)
  • getElementsByClassName (Mostly Used)
  • getElementByTagName
  • querySelector & querySelectorAll
What is event delegation in the context of the DOM, and why it is useful?

Event Delegation is basically a pattern to handle events efficiently. The process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes I would use:

  • element.setAttribute('id/class', 'id/class name')
  • element.removeAttribute('id/class', 'id/class name')
  • element.classList.add('class name')
  • element.classList.remove('class name')
